The Alignment Board

This Drag-and-Drop enabled Java OpenGL viewer makes it possible for users to superimpose aligned neurons from different fly brain samples.

The Cylindrical Alignment App

Available on Source Forge at Cylindrical Alignment App, this standalone application allows users to view aligned biomolecules such as DNA, RNA, and Amino Acid (protein) sequences. Based on a project I wrote while obtaining a Master's from Johns Hopkins, this viewer can handle inputs produced by such diverse tools as Gene Machine, ClustalW, and of course various flavors of BLAST (Basic Local Alignment Search Tool). The original viewer was written using Java3D, which is slowly being phased out of use. It was adopted by the Universiti Sans Malaysia to display search results for an award winning project, as shown here. Just scroll down and look for an image labeled "The MDG client default screen".
